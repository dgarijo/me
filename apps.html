Demos and applications

Below you can find the different demos and applications I have worked on the past. Note that some of them are just prototypes for showing the potential behind a model or exploiting the data exposed within a dataset.

LDP4ROs (2015)

(Under development) A prototype implementation to showcase how the Linked Data Platform standard can be used to manage Research Objects.

FragFlow (2013-201?)

(Under development) Main implementation of my thesis. It integrates a series of graph mining algorithms to find common fragments in scientific workflows, along with the means to filter the results and link them to the initial corpus of workflows.

Widoco (2013-2014)

WIzard for DOCumenting Ontologies, using the LODE Framework. The application allows creating the documentation of the ontology, adding rdf-a annotations and creating an intuitive folder structure. Also, it allows publishing the provenance of the vocabulary and evaluate it with the OOPS tool.

rohub.linkeddata.es (2013-2014)

(Contributor) Research Object landing page with all the Research Objects developed by the Ontology Engineering Group and following the Linked Data Principles.

vocab.linkeddata.es (2013-2014)

(Contributor) Vocabulary landing page with all the vocabularies developed by the Ontology Engineering Group and following the Linked Data Principles.

Latex2RO (2013)

Simple tool designed to help creating Research Objects (ROs) from LaTeX papers. Given a LaTeX file, the RO creator will extract its title and metadata to fill a partially structured HTML page, annotating it in RDF-a.

Wings Workflow Explorer (2012)

Prototype for exploring the workflows published in the Wings workflow platform according to the OPMW model.

Wings Workflow Exporter (2011)

Java project designed to publish workflows as Linked Data.

Aemet (2011)

Prototype application for showing the data from the Aemet meteorolical weather stations in Spain.

El Viajero (2011)

Demo for showing the geolocalized travel guides from the El País newspaper in a map. This demo won the Triplification Challenge, in 2011. An additional demo explores the content by author and resource.
